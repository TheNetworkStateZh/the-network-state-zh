# 关于民族国家

你可能认为你知道什么是民族国家，但你可能没有考虑过这个问题。在这个抽象的概念上戳一戳，乐趣就来了。你开始意识到民族与国家有多大的不同，确定谁有资格成为 "民族 "是多么棘手，我们围绕这一主题的现代术语是多么混乱，以及有多少其他人类组织模式代表着民族国家的潜在竞争对手。这种探索打开了通往网络国家的大门。

在这个过程中，你会遇到所有那些人们在学校里依稀记得的哲学家。你知道，洛克和卢梭，柏拉图和亚里士多德，无数无聊的读书报告的主题--他们中的许多人在这一章中都有表现。但他们在这里的存在与典型的 干巴巴 (opens new window) 的大学讲座不同，因为网络状态使政治科学成为一门应用科学，更像是政治技术。你在听的时候，意图是重复 (opens new window)。也就是说，就像加密货币给了美联储主席以外的人一个学习从 铸币税 (opens new window) 到 滞纳金 (opens new window) 的理由一样，加密国家给了开国元勋以外的人将政治理论付诸政治实践的能力。

但前提是你要理解这个理论，所以让我们深入了解一下。

# 什么是民族国家？

最明显的定义是，民族国家是世界上的一个地理区域，由一群我们称之为政府的人统治。当我们提到像美国和中华人民共和国这样的 "国家 "时，我们谈论的就是它。它是全球政治地图上一个标有旗帜的区域。

《大英百科全书》 (opens new window)提供了一个更精确的定义，即民族国家是一个 "以领土为界的主权政体"，它 "以一个认同自己为民族的公民群体的名义进行统治。" 后面这一点很关键，因为一个民族国家不仅仅是一个控制领土的政府。它应该是一个代表一个独特民族的政府，一个国家。

# 什么是民族国家体系？

约书亚-基廷在他的《看不见的国家》 (opens new window)一书中有一段关于民族国家体系的特殊性的精彩段落。他把这个系统比喻为一个有选择的俱乐部，有以下八条规则。

  • 规则1：一个国家是由所有国家共同商定的边界所界定的领土。
  • 规则2：一个国家必须有一个控制（或至少试图控制）在其境内合法使用武力的国家，以及有公民人口。
  • 规则3：地球上的每一个地方都必须被一个国家占领。
  • 规则4：地球上的每个人都必须是至少一个国家的公民。
  • 规则5：在纸面上，所有国家都有相同的法律地位--图瓦卢和中国一样有建国的权利，索马里和瑞士一样有建国的权利--即使它们在政治和经济上非常不平等。
  • 规则6：每个国家的人民同意是最好的，但不是必须的。一个国家内的暴政或事实上的无政府状态并不是失去俱乐部成员资格的理由。
  • 规则7：在某些情况下，一个或多个国家可以入侵或占领另一个国家，但不能消除其国家地位或重新划分其边界。
  • 规则8：目前现有的一组国家和它们之间的边界应尽可能保持不变，也就是说，俱乐部倾向于不接纳新成员。

基廷继续指出，这个俱乐部的规则是由联合国的机构和美国的军事力量支持的，数十亿人通过他们的政府对当前的世界秩序达成的协议是维护 "地图上的停滞 "的原因。

请注意，即使人们认为联合国是无效的，它也是这个系统的一个谢林点。没有别的东西有那么多的合法性，那么多的反向链接。

# 民族国家体系的假设

我们可以从一个不同的角度来描述民族国家体系的假设，这个角度可以让我们更容易理解这个体系和我们将在 下面几页 (opens new window) 介绍的网络国家体系之间的区别。

  • 实物第一。世界的物理地图是首要的。
  • 构成。从理论上讲，一个民族国家是由一个民族（人民）和一个行政实体（国家）组成。在实践中，一些 "民族国家 "实际上是多民族帝国，而一些国家则是无国籍国家。
  • 没有不为人知的土地。现代民族国家体系想当然地认为不存在 不为人知的土地 (opens new window)：物理世界的地图是完全已知的，以至于它可以被细分。
  • 没有无主地。这个系统还想当然地认为，没有无主地，没有无主的土地。除了少数例外，地球表面的每一块土地都由一个国家负责，而且只有一个国家。除了国际水域之外，大部分海洋也是这样划分的。
  • 自上而下的土地划分。完全可见的地图被雕刻成称为国的地理区域，其边界由经纬度精确划分。
  • 每个公民一个国。人们通常只是一个国的公民，改变公民身份的情况不多，而且大多数公民与他们的父母受同一个国的管理。公民身份的主要方法仍然是 血统制 (opens new window)，即出生制。
  • 合法性来自实际控制和选举选择。一个民族国家的合法性有几个来源。首先，国家需要有足够的暴力能力来实际控制它所声称的领土。第二，但实际上是次要的，国家的合法性应该是由其基础民族的支持和他们对普遍人权的尊重所决定的。(不幸的是，这是一个次要的观点，因为任何团体在事实上控制领土的时间足够长，最终 (opens new window) 都会 得到 (opens new window) 承认 (opens new window)）。理想的情况是，一个合法的国家反映其人民的意愿，同时也尊重个人的权利--让大众和少数人都有发言权。
  • 集中管理。国家的管理者，通常是行政部门和立法部门，在纸上写下法律，规定什么是强制性的和禁止性的。这些法律通常由司法机构解释，并由持枪的人执行。在民族国家体系中，每块土地都由一个国家管理，无论谁在那里。
  • 国家内部对暴力的垄断。每个国家通过警察部队维持其境内的秩序。违抗法律的公民将遭受越来越多的暴力，直到他们遵守法律，就像《侠盗猎车手》 (opens new window)一样。
  • 通过军事实现国际主权。原则上，国家不应该干涉其他国家的内部事务。在实践中，只有当一个国家有足够的能力通过警察、情报机构和军队来抵御国内和国外的对手时，它才能保持主权。
  • 通过双边和多边论坛的外交承认。各国可以相互签署双边协议，也可以得到联合国、世界贸易组织（WTO）、国际货币基金组织（IMF）和 20 国集团（G=20）等多边论坛的承认。外交承认既是一个政治问题，也是一个文书工作问题，缺乏承认会使一个国家和/或其公民被孤立。
  • 条约管理合作和约束。一套跨境契约试图管理国家间的互动并限制滥用，承诺诸如人权和行动自由--这些声明经常被藐视。
  • 美国的和平。最后，虽然不是一直如此，但目前民族国家体系的保障者是美国，也就是联合国的总部所在地，它声称要 "提供全球领导 "和 "捍卫基于规则的国际秩序"。所有其他国家必须希望这个基于规则的秩序的保障者 不会 (opens new window) 决定 (opens new window) 入侵 (opens new window)、监视 (opens new window)、制裁 (opens new window)、空袭 (opens new window) 或 (opens new window) 以 其他方式 (opens new window) 破坏 (opens new window) 它们 (opens new window) 的稳定。

这些涵盖了国家的六个基本部分：边界、人口、中央政府、国际主权、外交承认和国内暴力垄断。

# 作为术语的民族国家

理解 "民族国家 "这个术语需要我们将民族（具有共同血统、历史、文化或语言的一群人）与国家（他们的政府）区分开。它们是不一样的。

尽管 "民族 "经常与 "国家 "混为一谈 (opens new window)，但 "民族国家 "一词有两个词是有原因的。第一个词（nation）与 "natality "的 词源 (opens new window) 相同。它曾经表示一群有共同祖先的人。第二个词（国家）指的是管理这些人的实体，它指挥着警察和军队，并对国家所居住的地理区域拥有暴力垄断权。在某种意义上，民族和国家就像工厂里的劳工和管理层一样不同。前者是大众，后者是精英。

教科书式的民族国家就像日本，一个拥有共同血统和文化的单一群体（日本人）占据着一块明确划分的领土（日本的岛屿），并由一个明确的主权国家（日本政府）统治，该政府在某种意义上代表着人民（最初通过 神 (opens new window)，同时通过 国会 (opens new window)）。

# 微型国家和多国组织

这给了我们一个新的视角，让我们了解为什么像海岛这样的 微型国家 (opens new window) 行不通：它们从领土和政府开始倒退，而不是从一个民族和他们的文化开始前进。后一个过程是历史上民族国家出现的方式：一个国家是由一个民族建立起来的，以治理它，而不是反过来......尽管后来这个自我同一的国家经常 开始 (opens new window) 将 (opens new window) 其他国家 同化 (opens new window) 为其创始国，所以这是一个双向的过程。

尽管有双向性，但民族的蛋先于国家的鸡。从这个角度来看，比微型国家更好的术语其实是微型国家，因为除非它代表了一小部分结盟的人，否则它就不是一个微型国家。一个人自称是政府，只是一个微小的国家。俗话说，你和什么军队？没有国家，就没有军队，也就没有合法性。

在光谱的另一边是一个帝国，或多国。罗马帝国、奥斯曼帝国和苏维埃帝国包含许多国家和民族。

这个有利条件使我们可以纠正更多的词汇。例如，多民族（national）公司的概念有点名不副实；正确的说法是多国（state）公司（跨政体经营），以与多民族国家区分（在其边界内管理许多不同族群的事务）。

# 0-民族, 1-民族, N-民族

介于 0-民族的微型国家和 N-民族的帝国之间的是 1-民族的国家，这些政府的建立是为了在一个确定的领土上管理一个单一民族的事务。然而，虽然这种术语并不完全被废弃，但它有点老套了。这不是我们在今年倾向于谈论民族国家的方式。

首先，今天我们经常讨论多民族国家--像美国这样的多民族国家--它们实际上更 (opens new window) 像 (opens new window) 过去的 (opens new window) 帝国 (opens new window)，而不是经典的单民族国家。第二，许多人认为，在互联网时代，物理边界并不重要。第三，现代话语在更大程度上关注命题国家，共同的思想是组织原则，而不是共同的遗产。第四，也是最重要的一点，国家内部族群之间的冲突可能导致 内战 (opens new window)、大规模驱逐出境 (opens new window)、极权主义洗脑 (opens new window)、种族清洗 (opens new window)、强迫转换 (opens new window) 和 文化破坏 (opens new window)，这种过程最近导致 东帝汶 (opens new window) 和 南苏丹 (opens new window) 的形成。

稍后，我们将讨论网络国家如何解决这些问题，但这些都是民族和国家之间的区别已经失宠的（可以理解的！）原因。学者们不想无意中鼓励 分离主义 (opens new window) 或 种族隔离主义 (opens new window) 或 更糟 (opens new window) 的情况，以免人们认为除非政治实体（国家）代表世界上所有单一种族（民族）的成员，否则就不是一个真正的民族国家。

或者至少，他们不想在国内这样做。因为在涉及到这样的术语时，普通的美国人是有点精神分裂的。他可以很容易地理解，比如说，乌克兰人民希望摆脱俄罗斯帝国的束缚，或者西藏民族希望有自己的政府脱离中国国家，或者波斯人民希望将自己与伊朗的神权区分开来。但同一个人通常更怀疑英国应该退出欧盟，更不用说 "德克萨斯民族 " (opens new window)应该拥有自己的主权国家。

愤世嫉俗的人可能会说，国家愿望得到的播放时间与国家利益成正比；更愤世嫉俗的人可能会说，即使是 "民族利益 "这个词也是另一个错误的说法，因为鉴于美国国家统治着不止 一个民族 (opens new window)，它更像是 "国家的利益"。然而，这将我们引向一个关键问题：究竟什么是国家？

# 什么是民族？

这个问题曾经是非常重要的：哪些群体的重要性足以被称为国家，成为他们自己国家的候选人？这个问题很快又会变得非常重要，和 "什么是货币 "一样重要，原因也很相似：因为比特币、web3、元空间、远程工作、移动和互联网让人们比过去任何时候都更容易退出传统的安排，形成新的团体。但这些群体中哪些应该被视为 "国家"？

# 一个定义的方法

让我们从牛津的定义开始，通过他们的免费服务 Lexico：

因共同的血统、历史、文化或语言而结合在一起的一大批人，居住在一个特定的国家或地区。

从这个定义中，我们可以提取出以下属性。

  • 庞大的人群：必须有相当大的规模（10-10万以上？）
  • 团结：成员认为自己是同一群体的一部分。
  • 共同的血统：有共同的基因，比民族以外的人更多地相互通婚。
  • (或)历史：共同的过去，已经在彼此附近生活了一段时间。
  • (或)文化：共同的服饰、食物、举止、宗教和/或习俗。
  • (或)语言：共同的口语和/或书面语言。
  • "居住在一个特定的......地区 "：被发现在全球的一个特定区域。

这些作品中的每一件都可以被捅破。多大才是 "大"？我们如何衡量一群人是否团结一致？一个民族必须在某一特定领土上有多大的地方性，或者它可以是 游牧民族 (opens new window)？还有，为什么我们要在中间埋下一个复杂的 "或 "字 (opens new window)，共同血统、历史、文化或语言都在其中？我们的第一直觉是，民族的定义有点模糊，而我们的直觉是对的。

# 经验性的方法

为了使我们的讨论有根有据，我们来看看被称为民族的团体的具体例子：

  • 日本人。他们与定义完全一致。日本人曾一度拥有相当大的 帝国 (opens new window)，在美国（和巴西）也有日本侨民......但大多数有日本血统的人都生活在日本的岛屿上，说日语，受日本国家管辖，并生活在一个基本上是单民族的政体中。
  • 西班牙人。他们今天有一个民族国家，但在过去，他们有一个国际帝国，然后收缩，让他们在伊比利亚半岛上大部分是自己的。他们在全球范围内留下了 20 个 讲西班牙语 (opens new window) 的国家，但却不认为自己是西班牙民族国家的一部分。
  • 土耳其人。他们今天是一个多民族的国家，也是一个更大的帝国--奥斯曼帝国的继承者，具有 定义上的拜占庭历史 (opens new window)。
  • 以色列人。他们作为一个民族国家的地位随着时间的推移而改变。犹太人曾经是一个无国籍的民族，一个由共同的祖先和传统联合起来的散居群体，没有一个属于他们自己的土地或政府。然后，在人们的记忆中，他们建立了以色列国。(赫兹尔 (opens new window) 的作品是本书的主要灵感来源）。
  • 加泰罗尼亚人、库尔德人和巴勒斯坦人。当然，每一个西班牙、以色列或土耳其都有一个加泰罗尼亚、一个巴勒斯坦、一个库尔德斯坦--即一个自我认同为一个民族并认为其民族愿望被拒绝的群体。这些都是无国籍的国家，与民族国家不同，不一定赞同任何特定的事业。
  • 爱尔兰人。他们现在有一个 独立的爱尔兰 (opens new window)，但在英国统治下的许多年里却没有独立的爱尔兰。一个有争议的问题是 北爱尔兰 (opens new window) 应该是爱尔兰共和国的一部分，还是英国的一部分。
  • 台湾人。这个群体被某些方面 承认 (opens new window) 为一个民族，但绝不是所有方面。我们可以将其视为部分主权国家，对自己的国家和领土有一定程度的控制，但比他们希望的要少。
  • 美国人，新加坡人，还有法国人。这些国家试图从几个不同的民族群体的原材料中精心打造一个作为 "国家 "的共同身份，取得了不同程度的成功。事实上，从某种程度上说，美国人在这方面的努力非常成功--至少在一段时间内。美国人 (opens new window)、新加坡人 (opens new window) 和 法国人 (opens new window) 是明确提出的民族。
  • 中国人和印度人。鉴于每个国家的边界内都有大量的不同群体，这些大国并不是真正的单一民族国家。然而，这些不同的群体并不是最近才像 "火人 "营地那样彼此相邻而来。他们已经在一个共同的文明中并肩生活了几个世纪，过去的 (opens new window) 统一 (opens new window) 程度有大有小，所以这种分组比最近的多民族国家更 "林迪" (opens new window)，后者没有那么多长期的记录，更不用说殖民主义之后留下的完全任意的国家。有些人对这些实体使用了 文明国家 (opens new window) 这个术语。你甚至可以把它延伸到包括欧盟，尽管它更像是一个跨国官僚机构，而不是一个颂扬欧洲文明的实体。

而对于许多中东和非洲国家来说，这些国家根本没有真正反映出其背后的国家。这里的一个线索是地图上出现的水平线或垂直线，这些线并不反映帮助定义国家的有机物理（沙漠、山脉、河流）和文化（语言、婚姻、宗教）障碍。这些 "强加的国家 "中有许多是殖民帝国的临别礼物。

从这些例子中，我们已经可以看到相当多的变化：

  • 有国家的民族（日本、西班牙）。
  • 没有国家的民族（库尔德人、加泰罗尼亚人）
  • 拥有部分主权国家的民族（台湾）
  • 试图建立议题民族的多民族国家（美国、新加坡、法国）。
  • 强加的多民族国家，甚至没有一个命题来约束它们（许多 "国家 "是在欧洲殖民主义之后作为猎枪婚姻形成的）。
  • 多民族的文明国家，但有长期的文化纽带将其组成国家统一起来（中国、印度）。

通过参观这个话题，我们也看到，"什么是国家 "的问题仍然是热门话题，是第三条轨道，是脉搏的提升，是争论的开始。因为一个被赋予合法性的国家可以要求领土和建立政体，而一个被拒绝承认的国家仍然没有土地和国家，这个看似抽象的问题的利害关系再高不过。

# 一个哲学的方法

我们刚刚做了一些具体的例子。我们能否阐明一般原则，界定哪些群体应被视为真正的民族？从过去到现在，许多学者都对这个问题进行了探讨。

以下是他们观点的一个不完整的简述，部分摘自本纳的 章节 (opens new window) 和考夫曼的 评论 (opens new window)。首先，17世纪末和19世纪的思想家，在美国革命、拿破仑战争或 1848 年革命期间写作：

  • 卢梭 (opens new window)：如果一群人自愿同意接受同一管理当局的约束，他们就是一个民族。
  • 马克思 (opens new window)：民族是一个由大国支持的方便的团体，以破坏对手的稳定。关于共产主义，民族是一个可以领导获得政治优势的 群体 (opens new window)，也是一个可以 超越的界限 (opens new window)，以 联合无产阶级 (opens new window)。
  • 洛克 (opens new window)：如果两个群体对同一领土提出要求，那么更 "理性和勤劳 "的群体应被视为一个民族。
  • 约翰-斯图亚特-米尔 (opens new window)：如果一个群体同意同一个管理当局，并且有能力获得对 一块土地的控制权 (opens new window)，他们就应该被视为一个民族。然而，米尔的 效用概念 (opens new window) 战胜了同意 (opens new window)。
  • 黑格尔 (opens new window)：一个民族是由它的机构灌输一种共同的道德感而形成的。战争 (opens new window) 是对这种伦理责任的考验，它本身并不是邪恶的，而是无政府状态下国家间关系的一个自然条件。
  • JG-赫德尔 (opens new window)。如果一个群体有共同的语言和血统，它就是一个民族，这个概念被称为原始主义。此外，小民族应该独立于那些想把它们同化为不同语言的大民族。
  • JG-费希特 (opens new window)：和赫德尔一样，独立的语言和种族定义了独立的民族。此外，一个国家可以通过教育来建立一个民族，引导民众形成共同的文化和语言认同。
  • 欧内斯特-勒南 (opens new window)：一个民族是那些在过去有 "共同的荣耀 " (opens new window)和牺牲，并且 "在现在有意愿继续这些 "的民族。一个民族的存在是通过 "日常公民投票 "来体现的，它构成了一个民族目前的同意。
  • 欧内斯特-盖尔纳 (opens new window)：民族是共享（通过学校教育）语言、文化和特别适应现代社会的交流形式的民族。
  • 本尼迪克特-安德森 (opens new window)（Benedict Anderson）：民族只是社会结构，是想象中的社区，是基于 "印刷资本主义 "所驱动的语言联系。
  • 埃里克-霍布斯鲍姆 (opens new window)（Eric Hobsbawm）：民族必须有一个与当前国家的历史联系，一个长期建立的语言文化行政精英，以及一个被证明的征服能力。

这些定义既相互重叠又相互冲突。一些紧张的情况包括：

  • 原始主义与命题主义。民族可以是一个有共同祖先、文化和语言的群体，但也可以是纯粹基于思想和自愿联合的。
  • 规模与独特性。一个民族需要有足够的规模来保卫自己，所以它应该对国家成员采取一个广泛的定义。但它也需要避免被一个大规模的团体同化，以至于没有独特的文化可以捍卫。
  • 自决与外部赞助。一个民族在一定程度上是基于对民族的自我认同，但在实践中也需要能够提供现实世界的结果（是 "理性和勤劳"），并能吸引大国赞助的支持。
  • 想象中的社区与真实的语言文化联系。一个民族是一个想象中的社区和社会结构，但它需要分享足够多的相同语言和文化，以便可行地组装这个结构。

这些分歧意味着对于一个群体是否是一个民族还没有一个单一的测试，尽管在任何特定的情况下，人们可以通过诉诸不同的标准来提出或多或少有说服力的案例。然而，利用现代工具，我们也许能够理顺这种模糊性。在本章的后面，我们将介绍一种定义，民族的计算方法，以补充经验和哲学方法。我们将讨论这些关于民族起源的理论如何影响创业公司创始人的 "客户获取 "战略，或者在这个案例中，公民获取 (opens new window)。

但现在，什么是民族？也许它只是一个能够说服足够多的其他人认为它是一个民族的团体。

# 什么是国家？

也值得花时间研究民族国家定义的另一半：究竟什么是国家？

# 定义的方法

这个有用的 视频 (opens new window) 列举了状态的六个属性。

  1. 边界：一个明确界定的领土
  2. 人口：生活在该领土内的一个或多个民族
  3. 中央政府：制定法律的能力
  4. 国家主权：在理论上，对国内事务的控制不受其他国家的干涉
  5. 承认：其他国家在外交上的承认
  6. 国内暴力垄断：维持领土内秩序的能力

例如，一个处于内战中的失败国家就不合适，因为它无法阻止外国势力的干涉（第4项），也无法控制国内的暴力（第6项）。一个小国不算，因为它缺乏领土（第1项）和人口（第2项）。而像阿肯色州这样的美国行政分区也不算，因为它缺乏外国的承认（第5项）和相对于华盛顿特区的控制（第3项）。然而，一个分区有时可以 成为 (opens new window) 一个独立的国家。

# 比较的方法

比较一下怎么样？正因为它们经常被 混为一谈 (opens new window)，所以值得详细讨论国家与民族的 不同之处 (opens new window)。

  • 国家是一个政治和法律实体，而民族是一个文化、种族和心理上的认同。
  • 国家受法律和武力威胁的约束，而民族则受情感和语言/基因/文化的约束。
  • 国家是自上而下、等级森严的，而民族是自下而上、点对点的。
  • 而且，如上所述，国家有一个固定的领土，一个政府和对领土的主权，而民族通常有共同的语言、文化和/或祖先。

国家可能并不总是有一个单一的民族。库尔德人缺乏一个国家，而朝鲜人则被分成两个国家。相反，国家可能管辖一个或多个民族。英国国家管理着英格兰、威尔士、苏格兰和爱尔兰民族，而苏联国家则管理着100 多个不同的民族 (opens new window)。

虽然有些人认为民族和国家之间的区别是欧洲人固有的想法，但实际上这些概念在不同的语言中有不同的说法。

# 务实的方法

也许检验一个东西是否是真正的国家的最简单的标准是它是否是联合国大会的成员。它是否有足够的外交承认？它是否被其他我们认为是国家的实体认为是一个国家？一句话，它是否被承认？这一点很重要，因为即使是非常大的群体，如中国人和印度人，也比世界上其他国家的人多；社会生存能力是国家生存能力的必要条件。

关于这个话题的几本优秀书籍是《看不见的国家》 (opens new window)和《不在地图上》 (opens new window)，其中回顾了纳戈尔诺-卡拉巴赫、阿布哈兹、德涅斯特河左岸、北塞浦路斯、索马里兰、南奥塞梯和阿拉伯撒哈拉民主共和国、科索沃和台湾等边缘案例。这些实体中的每一个都有或多或少的内部国家性质，台湾是最合法的，但所有这些实体都缺乏某种程度的全面国家间承认--往往是由于一个强大的区域或全球对手。

在我们讨论联合国的时候，比 "联合国 "更好的名字可能是 "选定的国家"。毕竟，许多无国籍民族在联合国大会上没有席位，比如库尔德人、加泰罗尼亚人或西藏人。而许多有席位的国家更类似于多民族帝国，而不是单一民族国家。

# 哲学的方法

凯恩斯说："那些认为自己完全不受任何智力影响的实践者，通常都是一些不存在的经济学家的奴隶。" 意思是，如果你不知道你在运行什么智力软件，你可能是在无意识地运行它。所以，我们很难调查导致现代国家的许多思想家，因为我们并不总是了解他们影响的全部范围。

我们还是要尝试一下。这里有另一组必然不精确的总结，说明不同的政治理论家对国家的看法。

  • 柏拉图 (opens new window)：国家应该创造条件，使每个人都能养活自己并寻求善。
  • 亚里士多德 (opens new window)：所有社区都以某种善为目标，而国家是最高类型的社区，以最高的善为目标。
  • 洛克 (opens new window)：如果国家执行合同并作为私有财产的保证人，那么它就是合法的。
  • 卡莱尔 (opens new window)：国家应该由一个提供秩序的英雄来管理。
  • 施密特 (opens new window)：国家体现了明确的敌友之分。
  • 马克思 (opens new window)：国家是为了组织无产阶级反对统治阶级。
  • 凯恩斯 (opens new window)：国家应该进行干预，以平滑商业周期，支持充分就业。
  • 罗尔斯 (opens new window)：国家根据作为公平的正义理论，将社会产品和经济机会平等地分配给其自由公民。
  • 霍布斯 (opens new window)：国家拥有绝对权威，这个强大的利维坦使反社会的人以亲社会的方式行事。
  • 卢梭 (opens new window)：如果人们同意签订 "社会契约"，在契约中他们自我管理，最好不要把主权交给可能不一致的代表，国家就是合法的。
  • 萨缪尔森 (opens new window)：国家的目的是提供私人行为者无法提供的公共产品。
  • 李光耀 (opens new window)：国家应该让人民最大限度地享受自由，尊重家庭单位。国家应该包容多个民族，但 要求忠诚 (opens new window)。

# 国家战略和编程范式

再说一遍，这并不只是枯燥的理论。理解这些思想很重要，因为它们被实际存在的国家的创始人和领导人 隐含地 (opens new window) 或 明确地 (opens new window) 使用。

从计算机科学的角度来看，这些思想流派是类似于 编程范式 (opens new window) 的国家战略。也就是说，你经常可以从（比如）面向对象、函数式或命令式的角度解决 同样的 (opens new window) 问题。但某些问题用特定的范式更容易解决，而其他问题则变得 更难 (opens new window)。

对于这些不同的状态理论也是如此。此外，这些国家战略不是被孤立地使用，而是经常被融合在一个法律代码库中，就像不同的编程范式在一个公司的代码库中可以相互补充。

例如，卡尔-马克思的零和世界观使其很容易为一个拥有大规模红军的苏维埃国家消灭资本主义压迫者提供理由。相比之下，让-雅克-卢梭的著作并没有为使用武力本身提供多少理由，而是提供了一个协商一致的 共产主义乌托邦 (opens new window) 的愿景，它就在红军解放暴力的另一边。卡尔-施密特 (opens new window) 和 托马斯-卡莱尔 (opens new window) 是右派中大致相当的一对，施密特主张英雄对敌人使用国家武力，而卡莱尔则大谈因此而产生的丰饶秩序。

马克思和卢梭的失败模式是他们脱离了经济现实，因为他们没有考虑到自我利益。施密特和卡莱尔的失败模式是他们背离了政治现实，因为他们没有考虑到其他人的利益。但他们的治国策略曾经有足够的影响力，推动了世界历史上一些最强大的国家，所以我们需要理解他们，即使我们也必须抛弃他们。想想 PHP 是如何被许多工程师认为是一种 "糟糕 "的编程语言，但却 以某种方式引领了 (opens new window) 许多有史以来最流行的应用程序（Facebook、WordPress、Slack 等），你就会明白这一点。

也有可能完全朝另一个方向发展，让一个纯粹的契约型国家在隐含的哈耶克/洛克范式下运行，在没有马克思主义或施密特式国家叙事所提供的任何意义的情况下实现某种程度的财富最大化。这也有它的弱点，因为意义的真空可以被对手填补，其国家战略包括 不断地 (opens new window) 传教 (opens new window)；这就是为什么柏拉图式/亚里士多德式的国家叙事在优先考虑目的时有很好的意义。

各种治国策略的优点和缺点可以详细讨论，我们会回到这个话题。但是现在：在你设计你的理想国之前，你应该对其他人认为他们的理想国是什么，以及结果如何，有一些了解。

# 民族国家在地图上是什么样子的？

简单的答案是，一个民族国家是地图上的一个彩色圆点。但我们可以把这张地图看作是各种底层地图的叠加，显示出民族成员的位置--例如，讲该语言的人、具有共同等位基因的人和具有类似文化的人所居住的地方，叠加在国家的法律边界上）。

同样，日本是我们的典型例子。基本的地图都是一致的。大多数说日语的人，大多数有日本血统的人，大多数日元持有者，大多数神道教信徒，以及大多数具有日本文化的人，都居住在日本政府管理的日本岛屿上。

其他民族比这要混乱得多。

  • 一些民族围绕着一块领土零散地扩散，如巴尔干地区。
  • 有些民族已经扩散到世界各地，如以色列之前的犹太社区（今天在很大程度上仍然如此）。
  • 一些以前统一的民族由于历史原因被分割成不同的领土，如南北朝鲜。
  • 其他民族则由多个重叠的地图来定义，因为仅靠一个变量不足以划定它们。例如，如果你只是说所有说西班牙语的人都是西班牙民族的成员，你就会把各大洲数百万不认为自己是同一族群的人错误地归类。
  • 一些 "民族国家"，如印度尼西亚，有奇怪的边界--部分原因是它们实际上是多民族国家。
  • 一些 "民族国家"，如法国和英国，有着令人惊讶的全球分布，因为它们实际上是多民族帝国的遗迹。

一般来说，理想化的民族国家是一个特定群体--民族--的成员在物理上集中在地球表面的一个单一的边界集。这似乎很微不足道，但在本章后面，我们将在网络国家的背景下探讨物理上分散的政体。

# 现代民族国家是如何建立的？

关于民族国家如何建立的问题，有几个不同的角度。

# 历史的角度

第一个角度是思考许多国家是在什么时候根据大致相同的原则在同一时间成立的。我们可以定义历史上的几个关键时刻。

  • 二战和冷战（1945-1991）：今天的国家是在战后秩序的支持下建立的。二战后，在欧洲内部，大规模的人口转移创造了单一民族的国家。同时，在欧洲之外，西欧大国拥有的殖民地经历了 "非殖民化"，然后可以说是被苏联或美国分别以共产主义或资本主义的名义 "重新殖民化"。1991 年苏联解体后，又出现了一批独立国家。
  • 美国革命、法国革命、大分化（1776-1800 年代）。本尼迪克特-安德森等作家将现代意义上的欧洲民族主义的崛起归功于 19 世纪初法国大革命后的 "大分化"，而法国大革命又受到美国大革命的启发。
  • 30 年战争、西班牙/荷兰战争和威斯特伐利亚和约（1618-1648）。威斯特伐利亚和约结束了由宗教改革引发的新教徒和天主教徒之间的 30 年战争，并开创了具有有界领土主权的国家的概念，而不是天主教会的无界权威。
  • 地图制作 (opens new window) 和 印刷资本主义 (opens new window) 的崛起（15 世纪）。制图技术的兴起使准确的地图得以产生。今天，我们认为这是理所当然的，但如果没有好的地图，就没有超越地形的明确边界，只有一个主权国家的权力随着其领土与另一个主权国家的领土交融而逐渐减弱。
  • 古代时代。像中国和印度这样的文明国家可以追溯到古代，并且可以指出语言、文化和宗教习俗的某些连续性。
  • 史前时代。原始论者 (opens new window) 认为，支撑国家的民族早于书面历史，因为它们的语言、基因和文化纽带可以追溯到数千年前。换句话说，国家是自然发生的现象，更像是元素周期表，而不是社会结构，其边界在波特式的意义上是明显的。任何真正的现代民族国家在这个意义上都是在几千年前建立的。

重要的是，整个世界并没有在同一时间得到现代民族国家。例如，威斯特伐利亚的主权最初是在西欧内部建立的，但没有在西欧之外建立。欧洲民族国家应该尊重彼此的边界，至少在原则上是这样，所以他们 到国外 (opens new window)去征服其他地方。

但历史上的这些交界点仍然是思考民族国家建立的有用方法，其中一个或另一个隐约可见，取决于人们是否更关注 "民族"、"国家 "或 "民族国家 "的组合。

从实际的角度来看，显然你不可能在没有数千年历史的情况下建立一个像中国或印度这样的文明国家。但你也许可以从美国人的群体中提炼出一个新的 "民族"，如 摩门教 (opens new window)（est: 1830），或者从被沉闷的 爱沙尼亚苏维埃社会主义共和国 (opens new window) 压迫的同一个民族中构建一个令人印象深刻的新民族国家，如 "E"-stonia (opens new window)（est: 1991）。

# 赞助的角度

另一种方法是研究特定民族国家如何建立的细节。在研究足够多的这些历史时，我们会发现，民族独立并不仅仅是一个自决的问题，因为许多民族的命运并不完全由他们自己的努力决定。

例如，苏联是 "反帝国主义"，而这意味着让同情西方的资本家离开，让同情苏联的共产党人进入。法国人支持新生的美国国家，而这意味着在他们的英国对手的眼睛里戳了一个拇指。而今天的美国人鉴于与土耳其和沙特国家的联盟，对库尔德人或也门人没有太多意见，但鉴于他们与俄罗斯和中国的冲突，对乌克兰人、台湾人和维吾尔人却极为热心。

因此，为了实现其野心，一个无国籍的民族可能还需要一个赞助人，一种风险资本主义的大国。自决是不够的。

# 军事的角度

在人们的记忆中，许多国家都是通过武力建立起来的，但由于这些国家往往是通过武力建立起来的，所以有些人认为不通过武力建立新的国家是不可能的。

或者说是这样吗？他们说你 不能建立五角大楼 (opens new window)；他们并没有说你不能找到 邮局 (opens new window) 的竞争对手，或 出租车牌系统 (opens new window)，或 美国航空航天局 (opens new window)。他们反而直奔主题，我们没有可比性的最近的发现......或者说 我们 (opens new window) 有吗 (opens new window)？毕竟，五角大楼本身是由像你和我一样的人类在 1943 年 (opens new window) 建造的。印度、以色列和新加坡也同样分别于 1947 年、1948 年和 1965 年成立，并有自己的国防部的对应机构。

当然，对这个挑战还有其他解释。它可能意味着 "好吧，它发生在前一段时间，但我不认为五角大楼的形成过程可以重复。"或者也许 "组建一支大规模的新军队是不好的，因为那会破坏稳定。"甚至，"来吧，你不可能从头开始创立世界上最强大的军队。" 但回答这类问题会带来一个嵌入式的 "第二十二条军规 "。要么是考虑建立新国家的人一定想建立一支强大的新军队（危险！），要么是他们没有枪，会被那些有枪的人击垮（危险的天真）。

一个答案是，你不需要获得完全的主权，而是可以与现有的主权国家签订防御合同。事实上，这实际上是大多数 "真正的 "国家已经在做的事情--很少有真正拥有完全主权的国家，因为大多数国家以类似于美国或（如今）中国的方式将其国防承包出去。

另一个答案是，你可以就这个问题写一本书（也许我们需要再加一章），但对于一个基本的数字实体，在世界各地的物理分散，主要模式将是通过保密、假名、分散和加密的非暴力数字防御。以不同的方式，谷歌和比特币在没有庞大军队的情况下保护了数百万人的数字足迹。

# 为什么要建立民族国家？

另一种问法是：民族国家之前有什么？

简短的回答是，人们有不同的 身份堆积 (opens new window)。在欧洲，直到很久以后，民众才认为自己主要是 "法国人 "或 "德国人"。相反，他们根据自己的封建领主或地区（布列塔尼、普鲁士）或宗教（例如，新教/天主教）来思考自己。像天主教会这样的跨国实体也声称对所有信徒拥有统治权，无论他们身在何处，因此存在着教皇还是国王在任何特定管辖区拥有最终权力的问题。战争接踵而至 (opens new window)。

1648 年的 威斯特伐利亚和约 (opens new window) 解决了这些问题，被许多人认为是欧洲民族国家的起源。威斯特伐利亚和平以地图上的线条划分领土。在这样划定的每块领土上，都有一个代表该领土上人民的政府，有权代表他们行使武力。而这些 "主权 " (opens new window)国家应该彼此互不干涉。

在理论上，国家应该是减少暴力的一种创新。你留在你的轨道上，我留在我的轨道上。明确的主权国家将保持国内秩序，而国家主权原则将阻止来自国外的侵略。当然，结果并不完全如此；国内 (opens new window) 和 国家间 (opens new window) 的冲突仍然发生。但是，抽象的民族国家可能仍然比之前那个边界模糊的帝国和相互冲突的主权国家的时代要好。

# 民族国家是如何扩张和收缩的？

一个民族国家至少有四种扩张方式：

  • 人口上。通过繁殖或移民。当一个国家的出生人数多于死亡人数时，它就会增长。当一个国家的一个组成民族经历了人口增长，或者当它增加了移民（可能来自 不同的民族 (opens new window)），这个国家就会增长。请注意，国家的扩张和民族的扩张在这里可能是有区别的。
  • 地理上。通过征服（如伊凡雷帝对 俄罗斯的扩张 (opens new window)），通过收购（如 路易斯安那收购案 (opens new window)），或通过协议（如新加坡与马来西亚的 非自愿分离 (opens new window)）。
  • 经济上。通过贸易和市场的开放。这并不总是和平的：见 英国东印度公司 (opens new window)、斯梅德利-巴特勒 (opens new window) 和 鸦片战争 (opens new window)。
  • 意识形态上。通过教育和转化。革命时期的法国大量投资于教育所有公民 讲法语 (opens new window)，扩大自我认同的法兰西民族。同样，基督教、穆斯林和共产主义团体在传教方面也花费了巨大的努力。当然，虽然这种传教活动有的会扩大民族国家的支持基础（就像 毛泽东思想 (opens new window) 对中国的影响，以及 瓦哈比主义 (opens new window) 对沙特阿拉伯的影响），但其他类型的病毒性思想却以破坏稳定的方式跨越了国家和民族的界限。

# 国家如何影响民族，反之亦然？

民族国家的形成是双向的；民族创造国家，国家影响民族，依此类推。虽然民族必须是第一位的，但历史上许多最成功的民族国家通过从 文化吸引力 (opens new window) 到 强奸和掠夺 (opens new window) 等手段吸引邻近的（然后是不邻近的）人民加入建国人口。

在 加里波第 (opens new window) 之前，只有大约 2.5% 的 "意大利人 " (opens new window)讲 (opens new window)我们现在所知的意大利语，但当时是意大利的佛罗伦萨方言。同样，在法国大革命之前，只有不到 50% 的法国人 (opens new window) 讲 (opens new window) 今天的官方法语品种。而在俾斯麦 统一德国 (opens new window) 之前，普鲁士和奥地利之间一直存在着 竞争 (opens new window)（"德国二元论" (opens new window)），争夺究竟应该如何以及是否应该形成一个 "德国"。

一个相关的现象是政治边界和民族文化之间的反馈循环。三八线 (opens new window) 在韩国文化中并没有预先存在的历史意义，但在朝鲜战争之后，新的 "北朝鲜 "和 "南朝鲜 "群体之间的通婚率骤然下降。这种状况已经持续了 70 年；持续的时间越长，两个群体之间的文化差距就越大。

这种坚硬的政治边界与过去的自然物理边界（如河流、山脉和沙漠）的作用基本相同。它们阻碍了等位基因和文化的传播，从而促进了国家形成的动力。在政治/领土和语言/遗传/文化之间存在着一个反馈回路。

# 什么不是民族国家？

什么不是民族国家？我的意思不是说香蕉不是民族国家这种琐碎的东西。我的意思是，在物理世界中，还有什么大规模的组织人民的方式不是民族国家？

换句话说，要理解什么是东西，我们需要理解它不是什么。我们生活在一个民族国家的世界里，所以将不同的东西概念化是很困难的。理想的反例是接近但不完全是这样的东西。这里有几个例子。

  1. 像苏联这样的多民族帝国不是传统意义上的民族国家，因为它们的疆域内有不止一个民族。
  2. 像库尔德人这样的无国家民族也不是民族国家，因为他们缺乏正式承认的领土和政府。
  3. 像天主教会这样的跨国运动也不是民族国家，因为所有信徒的集合并不包含在它所管理的一个领土国家内。(教会确实有 梵蒂冈城 (opens new window)，但那就像英国皇室一样仅有仪式性）。
  4. 像 ISIS 这样的恐怖组织跨越边界活动，有时还夺取领土，但不被认为是国家，因为他们缺乏外交承认（由于他们犯下了令人发指的罪行！）。也就是说，苏联共产党人是他们 那个时代的 ISIS (opens new window)，他们只是坚持了 16 年才让 FDR 承认他们 (opens new window)，所以只要有足够的 毅力 (opens new window)，这种称呼就 可以改变 (opens new window)。
  5. 像 罗姆人 (opens new window) 和 马赛人 (opens new window) 这样的游牧部落不是民族国家，因为他们在国家之间迁移。事实上，大多数人类曾经都是这样生活的，耕种/当兵是相对最近的创新，随着数字游牧的出现，我们可能会回到类似的状态。
  6. 像谷歌这样的多辖区公司，其服务器上的人比大多数国家都要多，而且确实控制着用户生活的巨大部分，如他们的信息和余额。然而，他们是迈向我们的网络国家概念的过渡形式，因为他们的用户缺乏国家的民族意识，他们的治理缺乏我们对国家的期望。
  7. 像日本人或亚美尼亚人这样的民族散居地并不是民族国家。他们可能有商业区，在这些地区有某种程度的社区组织，但他们只是一个民族的卷轴，而不是一个完整的国家，当然也缺乏一个完整国家的属性。
  8. 像阿富汗的 普什图族和哈扎拉族 (opens new window) 这样的地方部族并不是民族国家。他们是一个失败国家中的不同民族。
  9. 像欧盟、世贸组织或国际货币基金组织这样的超国家实体也不是民族国家，就其跨辖区的影响而言，它们更类似于天主教会。

# 哪些技术发展是现代民族国家体系的基础？

我们通常不会认为地图制作、印刷和火器是新奇的活动，因为这些基础技术是在许多代以前发明的。但是，它们都是我们现代国家边界概念的基础，在这些国家中，持枪的人执行着书面法律。

  1. 绘制地图。只有在拥有世界地图的情况下，我们才有可能拥有一张将其划分为民族国家的世界地图。你不必成为一个制图行家，也知道在 1492 年，当哥伦布航行在蓝色的海洋中寻找可以进行贸易的印度时，这样的地图并不存在。带有 "此处有龙 "的 "古老 "地图必须经过艰苦的制作。在现代 GPS 之前，围绕地图制作有一个巨大的技术堆栈，包括罗盘、望远镜和天体导航。
  2. 印刷。不仅仅是印刷厂，整个印刷资本主义的实践都有助于民族国家的产生。就像 Facebook 和谷歌希望每个人都在互联网上，从而扩大他们的客户群一样，15 世纪的新商业印刷商希望每个人都说同样的语言，从而使他们的商品销售最大化。
  3. 火器。"上帝造人，但山姆-柯尔特让他们平等"。封建主义是由身穿闪亮盔甲、手持重剑的骑马骑士来执行的；枪支改变了这一点。其他人已经写了关于向枪支时代过渡的文章，但总之，枪支降低了身体不平等的重要性。任何有枪的男人（或有枪的女人）都可以杀死任何其他男人，即使射手年老体弱，被射者是兰斯洛特爵士本人。火器（以及弩和大炮）的出现破坏了封建等级制度的稳定；强壮的右臂突然不如强壮的左脑，因为生产火枪所需的技术和供应链突然变得更有价值。枪帮助催化了从封建等级制度到民族主义共和国的过渡，并帮助促进了美国和法国革命的 "共和 "理想。

因此：地图制作、印刷和火器的结合为后威斯特伐利亚时代的民族国家奠定了基础，在那里，地图划定了边界，印刷的文件确立了法律，而一个拿着枪的人对越过这些边界或违反法律的人开枪。
